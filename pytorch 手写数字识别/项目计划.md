## 手写数字识别 项目实战
- 通过 pytorch 构建的模型, 实现根据手写输入数字, 输出数字文本
- 训练数据使用 https://github.com/DeepTrackAI/MNIST_dataset/(训练集60000张, 验证集10000张)
- 技术框架采用 torch, pygame
- pygame 实现手写输入
- torch  实现模型构建

## 神经网络架构设计
- 输入 28x28 单通道灰度图
- CNN层 单通道输入 3x3卷积核 输出16张特征图
- 池化层 2倍缩小
- CNN层 16通道输入 3x3卷积核 输出32张特征图
- 池化层 2倍缩小
- CNN层 32通道输入 3x3卷积核 输出64张特征图
- 全连接 64 * 3 * 3 输入 320 输出
- 全连接 320 输入 320 输出
- 全连接 320 输入 160 输出
- 全连接 160 输入 40  输出
- 全连接 40  输入 10  输出
- 输出 0 - 9 中取最高输出位作为最终结果

## 训练策略
- 训练20轮
- 每批次16张图
- 步长为0.0001
- 优化器采用Adam
- 损失函数采用CrossEntropyLoss
